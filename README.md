# An0mCorpses
A simple lootable corpses pluginYou can use this dependency referencing https://an0m.dev/maven

Example:
```xml
<repositories>
    <repository>
        <id>an0m-repo</id>
        <url>https://an0m.dev/maven/</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>dev.an0m</groupId>
        <artifactId>An0mCorpses</artifactId>
        <version>LATEST</version> <!--Or just use a specific version-->
    </dependency>
</dependencies>
```
